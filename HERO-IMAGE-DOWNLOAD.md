# Hero Background Image - Download Instructions

## Required: Hero Background Image

**Image:** Person Jumping on Trampoline with Child
**Photographer:** Brett Wharton
**Source:** Unsplash (Free to use)

### How to Download:

1. Visit this URL: https://unsplash.com/photos/fJBY2w66mLo
2. Click the "Download free" button (top right)
3. Save the file as `hero-bg.jpg`
4. Place it in: `C:\Dev\born2fly.com\assets\images\hero-bg.jpg`

### Alternative Direct Download:
https://images.unsplash.com/photo-1655842556432-ece48a4f2c1a?auto=format&fit=crop&w=1920&q=80

## That's it!

Once you place `hero-bg.jpg` in the `assets/images/` folder, your hero section will display this dynamic trampoline image as the background.

The current placeholder will be automatically replaced.
